# SmartTaskManager
Smart Task &amp; Productivity Manager is a C# application developed as part of the ITS203 Object-Oriented Design and Programming unit. The application helps users organize and manage daily tasks by allowing them to create, view, update, delete, and track tasks efficiently. 
Features
Add Tasks
View Tasks
Delete Tasks
Mark Complete
OOP Concepts
Exception Handling
